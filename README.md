# 💣 Bomberman Unity Game

A Bomberman-style 2D multiplayer/solo game built in Unity. Place bombs, destroy obstacles, collect power-ups, and be the last player standing!

## 🎮 Features

- Classic Bomberman gameplay mechanics
- Bomb placement with timed explosions
- Destructible and indestructible walls
- Power-ups system (e.g. extra bombs, blast radius, speed boost)
- multiplayer support
- Simple pixel-style or 2D grid-based level design
- Responsive controls and smooth movement
---
## 🕹️ Controls
### Player 1
- Move: WASD
- Place Bomb: Space

### Player 2 (if available)
- Move: Arrow Keys
- Place Bomb: Shift
---
## 💥 Gameplay
1. Move around the grid-based map
2. Place bombs to destroy blocks and opponents
3. Avoid your own explosions
4. Collect power-ups to become stronger
5. Be the last player alive to win
---
## 🧩 Power-ups
- 💣 Extra Bomb – increases number of bombs you can place
- 🔥 Blast Radius – increases explosion range
- ⚡ Speed Boost – increases player movement speed
---
## 🛠️ Built With
- Unity Engine (C#)
- 2D Tilemap system
- Custom player & bomb logic scripts
---
## 🚀 How to Run
1. Clone or download the project
2. Open the folder in Unity Hub
3. Open the main scene (e.g. `GameScene`)
4. Press Play in the Unity Editor
